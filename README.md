## Description

Simple feed aggregator, which collect RSS feeds and then watched for a new posts.

## Stack

- GitHub Actions (CI)
- JavaScript
- HTML
- CSS
- Webpack
- ESLint
- Axios
- i18next
- Yup
- on-change
- Lodash

## How to install

`git clone https://github.com/y-brs/frontend-project-11.git`

`make install`

`make start`
